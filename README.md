# calculator

node v15.10.0で動作しています

## How to run
1. 初回時のみ `npm i`を実行
2. `npm start`を実行し、`http://localhost:3000/`をブラウザで開く
3. storybookを開く場合、`npm rub sb`を実行し、`http://localhost:6006/`をブラウザで開く

## 使ったもの
- [Layoutit Grid](https://grid.layoutit.com/)
